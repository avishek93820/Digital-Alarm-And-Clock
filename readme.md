# ⏰ Digital Clock & Alarm

A simple web-based digital clock with alarm functionality.

## 🚀 Features
- Real-time digital clock (12-hour format)
- Multiple alarms
- Snooze (5 minutes)
- Enable/Disable alarms
- Delete alarms
- Persistent storage using localStorage
- Custom alarm tones

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📂 Project Structure
digital-clock-alarm/
│── index.html
│── style.css
│── script.js
│── assets/


## ▶️ How to Run
1. Download or clone the repo
2. Open `index.html` in browser
